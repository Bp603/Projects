This Project is About Restaurant management system
There are panals 
Admin: Admin can add food category, Food item, manage Staff, manage payment, Review Feedback, and manage table.

Waiter: Waiter can create order, book tables, take feedback, make payment. 

Cook: Cook can confirm order detail or food item which is done in kitchen.

To run this project you have to connect with database MYSQL.
